# Charte graphique documentaire — Présentation exécutive SIG-FDSU RDC

## Intention

Une identité institutionnelle sobre, lisible et contemporaine : elle traduit la confiance, la maîtrise territoriale, la décision publique et l’ouverture vers l’avenir.

## Palette

| Rôle | Nom | Couleur |
|---|---|---|
| Couleur principale | Bleu institutionnel | `#0B3D6E` |
| Fond profond | Bleu nuit | `#071A2F` |
| Accent décision | Or | `#D9A441` |
| Données / information | Bleu clair | `#38BDF8` |
| Impact positif | Vert | `#10B981` |
| Attention | Ambre | `#F59E0B` |
| Risque / alerte | Rouge | `#DC2626` |
| Texte principal | Ardoise | `#1E293B` |
| Fond clair | Ivoire | `#F8FAFC` |

Les couleurs ne doivent jamais être le seul vecteur de signification : tout état porte aussi un libellé et, lorsque nécessaire, une icône.

## Composants

- **Encadré message clé :** fond bleu nuit, texte blanc, filet or.
- **Encadré preuve :** fond bleu très clair, source visible.
- **Encadré limite :** fond ambre pâle, icône attention et formulation explicite.
- **Tableau :** en-tête bleu institutionnel, lignes alternées très claires, aucune bordure décorative excessive.
- **Figure :** titre court, légende, source, date et numéro.
- **Schéma :** fond transparent ou clair, une idée par figure, lecture de gauche à droite ou haut vers bas.

## Règle de composition

Chaque double page porte une idée directrice :

1. un titre qui affirme l’idée ;
2. un message clé ;
3. une visualisation, un chiffre ou une comparaison ;
4. une conséquence pour la décision.
