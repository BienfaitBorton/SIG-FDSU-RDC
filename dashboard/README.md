# Dashboard SIG-FDSU RDC

Ce dossier contient l’architecture de l’interface graphique du SIG-FDSU RDC.

## Objectif

- page d’accueil professionnelle
- menu latéral
- barre supérieure
- zone centrale de contenu
- navigation entre les modules

## Structure

- `index.html` : page principale
- `styles.css` : styles visuels
- `app.js` : navigation client-side entre les modules

## Utilisation

Ouvrir `dashboard/index.html` dans un navigateur pour visualiser le squelette de l’interface. Aucune connexion à la base de données n’est effectuée.
