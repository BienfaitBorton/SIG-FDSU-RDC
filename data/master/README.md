# Référentiel National des Actifs FDSU (store local)

`registry.json` est le store versionné du Référentiel National.

Bootstrap initial depuis `data/reports/fdsu_nomenclature.json`, lui-même dérivé de la
nomenclature officielle unique :

`data/raw/FDSU Structure code Territoire zones.xlsx`

Ne pas éditer manuellement sauf opération de gouvernance contrôlée.
